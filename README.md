# star-wars-wiki
Simple Star Wars wiki built with three different frameworks - Angular (Angular Material), Ionic (Angular, Cordova), VueJS.
* Angular
  - admin module
  - login page
  - films page
  - characters page
* VueJS
  - admin module
  - login page
  - register page
  - films page
  - characters page
* Ionic (Angular, Cordova) (tested on Android)
  - login page
  - films page
  - characters page
  - fingerprint authentication
